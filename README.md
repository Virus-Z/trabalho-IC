# Classificação de Dados Astrofísicos com Machine Learning 🚀

## 📌 Sobre o Projeto
Projeto de pesquisa (Iniciação Científica) focado em aplicar técnicas de Ciência de Dados e Machine Learning para processar, limpar e modelar grandes volumes de dados espaciais (arquivos densos como `.h5` e `.csv`).

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python
* **Manipulação de Dados:** Pandas, NumPy, h5py
* **Machine Learning:** Scikit-Learn

## 📊 O Problema e a Solução
O desafio principal consiste em ingerir e realizar a Análise Exploratória (EDA) de grandes conjuntos de dados brutos da astronomia. Através de algoritmos de Machine Learning, o objetivo é criar modelos preditivos capazes de classificar estruturas espaciais, traduzindo dados complexos em insights estruturados.
